## react-native-fyber

A react-native module for Fyber offer walls

The offer wall has an imperative API.

Fyber iOS SDK v9.0.0 is already packaged up to save time and pain! 

### Installation

1. `npm i squaretwo/react-native-fyber-offerwall#v0.0.1 -S`
